# Выпускная квалификационная работа: Разработка хранилища данных информационной системы пространственного развития территории

## Описание

Данный проект представляет собой разработку прототипа хранилища данных для информационной системы пространственного развития территории с использованием PostgreSQL/PostGIS и модели Data Vault.

## Цели проекта

* Проектирование архитектуры хранилища данных (RAW → Data Vault → Data Marts)
* Реализация ETL-процессов
* Выполнение пространственного анализа
* Обеспечение качества данных

## Используемые технологии

* PostgreSQL / PostGIS
* Python
* Docker

## Структура проекта

* src/ – исходный код и ETL-скрипты
* db/ – скрипты базы данных
* sample_data/ – входные данные
